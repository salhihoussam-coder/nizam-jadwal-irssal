# نظام تسيير جدول الإرسال لمتوسطة

تطبيق ويب متقدم لإدارة وتسجيل المراسلات الصادرة من المؤسسة التربوية، مبني على أساس البرمجة الكائنية التوجه (OOP).

## الميزات الرئيسية

- ✅ نظام دخول آمن مع صلاحيات مختلفة
- ✅ تسجيل وتعديل المراسلات الصادرة
- ✅ جدول تفاعلي مع بحث وترتيب
- ✅ أرشيف إلكتروني للمراسلات والمرفقات
- ✅ تقارير شاملة (يومي، شهري، سنوي)
- ✅ تصدير إلى PDF و Excel
- ✅ لوحة تحكم مع إحصائيات
- ✅ واجهة عربية متجاوبة
- ✅ معمارية OOP احترافية

## المتطلبات

- PHP 7.4 أو أعلى
- MySQL 5.7 أو أعلى
- خادم ويب (Apache, Nginx)
- المتصفح الحديث
